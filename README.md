# stat_558_proj_2
Project 2 for the Summer 2026 online course Stat 558 at North Carolina State University.

This includes both a "static" quarto for R page by which I obtained familiarity with the data set.

The data, from https://www.kaggle.com/datasets/keonim/nfl-game-scores-dataset-2017-2023, includes all final scoring information for every United States National Football League (NFL) game from 2010 - 2025.

The data was also used to build an R shiny applet to analyze and subset the data per the user's request.

Note the app is live at the site https://bgsandor.shinyapps.io/footballapp/
