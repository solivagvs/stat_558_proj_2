We create one more column indicating the victor in a given game.

football <- add_column(football, "Winner" = "")

for (i in 1:length(football$Winner)){
  if (football$AwayScore[i] > football$HomeScore[i]){
    football$Winner[i] = football$AwayTeam[i]
  } else if (football$AwayScore[i] == football$HomeScore[i]){
    football$Winner[i] = "TIE"
  } else {
    football$Winner[i] = football$HomeTeam[i]
  }
}

We will then add a Date column, combining the GameDate “prefix” and the Season “suffix” to the appropriate date type format. Note there are several entries with an NA designation and thus won’t correctly parse.

football <- football |>

football$GameDate <- mdy(football$GameDate)

For an example, Green Bay won 122 of the games for which it was the home team, out of a total number of 207 victories, yielding a proportion of \(\frac{122}{207} \approx 0.589\). Note in a binomial experiment with probability of success \(p = 0.5\), the probability of at least 122 victories in 207 games is \(P(X \geq 122) =\)pbinom(q = 122, size = 207, p = 0.5, lower.tail = FALSE)\(\approx 0.0041\), suggesting a significant home field advantage.
